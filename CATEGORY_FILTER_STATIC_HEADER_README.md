# NutsNews iOS Static Header Category Filter Update

This update changes the main feed header to stay fixed while the article cards scroll underneath it.

Changes:

- Removes the large native `Nuts News` navigation title
- Removes the previous descriptive intro copy
- Adds a centered, thinner, more elegant amber `NutsNews` title
- Keeps the title/category banner static while scrolling
- Always includes these category chips: `All`, `Uplifting`, `Community`, `Wellness`, `Achievement`
- Adds any other loaded article categories after the default chips
- `All` shows every card
- Other categories filter cards by the selected category
